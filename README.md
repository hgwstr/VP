LatLon — это простое Android-приложение на языке Kotlin, которое определяет текущее местоположение пользователя (широту и долготу) с использованием Google Play Services и библиотеки FusedLocationProviderClient. Приложение запрашивает разрешение на доступ к геолокации и отображает координаты на экране.

Функциональность

Запрос разрешения на доступ к местоположению у пользователя.
Получение текущих координат (широта и долгота) с помощью FusedLocationProviderClient.
Отображение текущего местоположения на экране.
Обновление координат по запросу пользователя.

Требования

Android SDK версии 21 (Lollipop) и выше.
Подключение Google Play Services для использования FusedLocationProviderClient.
Для работы приложения требуется разрешение на доступ к геолокации: ACCESS_FINE_LOCATION.

Установка

Клонируйте репозиторий:
git clone https://github.com/your-repo/latlon.git
Откройте проект в Android Studio.

Добавьте необходимые зависимости в файл build.gradle:

implementation 'com.google.android.gms:play-services-location:21.0.1'
Синхронизируйте проект с Gradle.

Использование

При запуске приложения оно запросит разрешение на доступ к вашему местоположению. Если разрешение предоставлено, вы сможете увидеть ваши координаты на главном экране.
Нажмите кнопку Get Location, чтобы обновить координаты вручную.

Основные классы

MainActivity
Главный экран приложения, который инициализирует FusedLocationProviderClient и отвечает за запрос разрешений и получение местоположения.

LocationScreen
Компонент, который отображает широту и долготу на экране, а также предоставляет кнопку для обновления местоположения.

Основные функции

getLocation(): Получает последнее известное местоположение пользователя и обновляет UI.
updateUI(lat: Double, lon: Double): Обновляет UI с новыми значениями широты и долготы.

Разрешения
Приложение требует следующего разрешения для работы:

<uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
При отсутствии разрешения оно будет запрашиваться у пользователя.

Темы
Приложение использует тему Material3, которая настраивается в файле LatLonTheme.
